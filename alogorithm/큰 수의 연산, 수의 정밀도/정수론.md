# lcm, gcd
유클리드 최대공약수 알고리즘   
2개의 자연수의 최대공약수는 큰 수에서 작은 수를 뺀 수와 작은 수의 최대공약수와 같다.   
분할정복 (주로 nC2의 시간복잡도 이후에 생각하게 되는 방법이다.)   
```python
def gcd(a,b):
    if a==0:
        return b
    else:
        return gcd(b%a, a)

def lcm(a,b):
    return int((a*b)/gcd(a,b))
```

for i in range(1, int(math.sqrt(N))+1):

## 진수 구하기 (n진수를 m진수로)
```python
def sol(n,m):
    s = 0
    while True:
        if n//m==0:
            s+=n
            break
        s+=n%m
        n=n//m
    return s

```
